# chat2ide

Self-hosted web and mobile terminal for long-running Codex CLI sessions.

Read the full README in:

- [English](README.en.md)
- [简体中文](README.zh-CN.md)

In short, this repository runs Codex CLI as real server-side PTY processes and lets one authenticated user control them from a browser or phone. It is for a trusted personal server, not for multi-user IDE hosting.
